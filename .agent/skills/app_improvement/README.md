# App Improvement Skill

## 使用方法

### 基本的な実行

```bash
python .agent/skills/app_improvement/agent.py
```

### スキル呼び出し

```
/app_improvement
```

### 機能指定の実行

```
/app_improvement --focus ui
/app_improvement --focus performance
/app_improvement --focus testing
```

## 主な機能

### 🔍 自動分析

- フロントエンド構成の分析
- バックエンド品質の評価
- パフォーマンスボトルネックの特定
- テストカバレッジの確認
- ユーザー体験の評価

### 🎯 改善領域の特定

- UI/UX改善点の洗い出し
- パフォーマンス最適化項目
- 機能拡張の機会
- コード品質改善項目
- セキュリティ強化ポイント

### 📋 改善計画の生成

- 優先順位付けされた改善項目
- 工数見積もり
- 段階的実行計画
- リスク評価

### 🔧 自動実行

- 即時実行可能な改善項目の自動適用
- コードの安全な修正
- テストの追加
- 設定ファイルの最適化

## 実行例

### 完全自動改善

```
スキルを呼び出すだけで、以下を自動実行：
1. 現状分析
2. 改善点の特定
3. 優先順位付け
4. 即時実行項目の適用
5. レポート生成
```

```
UI改善のみを実行：

```

## 設定オプション

### 自動実行設定

```json
{
  "auto_execute_immediate_actions": true,
  "generate_reports": true,
  "backup_before_changes": true,
  "max_effort_hours": 40
}
```

### フォーカス領域

- `ui`: UI/UX改善に特化
- `performance`: パフォーマンス最適化に特化
- `testing`: テストカバレッジ向上に特化
- `features`: 機能拡張に特化
- `quality`: コード品質向上に特化
