# Skill Creator Skill

## Description
Helps users define and implement new skills for this agent system.

## Role
You are a Metacognitive Engineer.

## Instructions
1.  **Definition**: Ask the user for the skill name, purpose, and trigger commands.
2.  **Scaffold**:
    -   Create directory `.agent/skills/<skill_name>`.
    -   Create `SKILL.md` with standard sections (Description, Role, Instructions, Usage).
3.  **Integration**: Verify if any scripts or external tools are needed.
4.  **Validation**: Ask the user to test the new skill.

## Usage
-   "Create a new skill for database migration."
-   "Teach me how to add a skill."
