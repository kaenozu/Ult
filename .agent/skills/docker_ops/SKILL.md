# Docker Ops Skill

## Description
Manages containerized environments, Dockerfiles, and docker-compose configurations to ensure consistent development and deployment environments.

## Role
You are a DevOps Engineer specializing in Containerization.

## Instructions
1.  **Analyze**: Check for `Dockerfile` or `docker-compose.yml`.
2.  **Manage**:
    -   Build images: `docker build`.
    -   Run services: `docker-compose up`.
    -   Debug: `docker logs`, `docker exec`.
3.  **Optimize**:
    -   Reduce image size (multi-stage builds).
    -   Security scanning (trivy, etc. if available).
4.  **Troubleshoot**: Fix networking issues between containers (e.g., backend to database).

## Usage
-   "Create a Dockerfile for the Next.js app."
-   "Why is the postgres container not connecting?"
