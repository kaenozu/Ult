# Trader Pro 開発マスター (Trader Pro Dev Master)

このスキルは、プロジェクトの全機能、UIの日本語化、およびデータの論理的整合性を「永続的」に維持するための最高指針です。

## 1. ファイル操作の鉄則 (Anti-Degradation)
*   **全機能継承**: ファイルを `write_file` または `replace` する際、既存の重要なロジック（日本語化、削除ボタン、特定のアクション呼び出し等）が失われないよう、必ず事前に `read_file` で現状を把握し、それらをすべて包含した内容を書き込む。
*   **不整合の即時検知**: 変更後にUIが英語に戻ったり、ボタンが消えたりした場合は、即座に「退行（デグレード）」と判断し、以前の最高品質の状態へ復元する。

## 2. データの論理的品質 (Quality Gatekeeper)
*   **異常値の監視**: 騰落率 100% 超え、資産の `NaN` 表示など、論理的にあり得ない数値がUIに露出することを防ぐ。
*   **スマート・データ管理**: 5年分データの IndexedDB 永続化と差分フェッチ（Delta Fetching）の仕組みを維持し、常に爆速なUXを提供する。

## 3. ユーザー体験の一貫性
*   **日本語ローカライズ**: すべてのUIラベル、メッセージ、AI分析理由は日本語で統一する（始/高/安/終、買い/売り、決済など）。
*   **初期状態の配慮**: ウォッチリストが空の場合は勝手に銘柄（トヨタ等）を表示せず、適切な案内（プレースホルダー）を出す。

## 4. 安全な運用
*   **自己保護**: サーバープロセス管理時に Gemini CLI 自身を終了させない。
*   **Git運用**: 常に安全なブランチで作業し、整合性を確認してからマージする。
