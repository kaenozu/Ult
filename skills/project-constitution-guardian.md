# プロジェクト憲法ガーディアン (Project Constitution Guardian)

このスキルは、Trader Pro開発の北極星である `GEMINI.md` の精神を全てのコードと対話に反映させるためのものです。

## 1. 言語の徹底
*   **日本語第一主義**: AIエージェント自身の発言、ソースコード内のUI文字列、およびJSDocの一部は日本語で記述すること。
*   **正確な翻訳**: `i18n` ストアとの整合性を保ち、意味の通じない直訳を避ける。

## 2. 技術的整合性
*   **App Router構造の遵守**: 全てのページ遷移とデータ取得は、Next.js 16の App Router パターン（`page.tsx`, `layout.tsx`, `error.tsx`等）に従うこと。
*   **Tailwind CSS v4 の活用**: インラインスタイルを避け、ユーティリティクラスによる一貫したスタイリングを徹底する。

## 3. レビュー基準
*   **依存関係の最小化**: `package.json` にないライブラリを勝手に導入しない。
*   **デグレードの禁止**: 新機能追加時に、既存の `TechnicalIndicatorService` や `MarketDataService` の既存機能を破壊しない。

## 4. メンテナンス
*   プロジェクトの構造が大きく変わった場合は、速やかに `GEMINI.md` を更新し、自身のコンテキストを最新の状態に保つこと。
