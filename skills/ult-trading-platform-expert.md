# ULT Trading Platform Expert スキルモード

ULT Trading Platformの開発を効率化するための専門的なKiloスキルモードです。複雑な金融取引システムの開発、保守、改善に特化しています。

## スキルモードの特徴

### 1. 取引システム開発の専門知識
- **リアルタイムデータ統合**：複数取引所のWebSocketデータフィード
- **AI予測分析**：機械学習モデルによる価格予測
- **センチメント分析**：ニュースとソーシャルメディアの感情分析
- **自動リスク管理**：動的なポジションサイジングとリスク制御
- **アルゴリズム取引**：低レイテンシーの注文実行
- **バックテスティング**：過去のデータに対する戦略評価
- **ポートフォリオ最適化**：現代ポートフォリオ理論に基づく資産配分

### 2. 技術スキル
- React/TypeScript/Node.jsのフルスタック開発
- WebSocketとREST APIの統合
- 機械学習モデルの実装
- リアルタイムデータ可視化
- 高パフォーマンスアルゴリズムの開発
- セキュリティとコンプライアンス

### 3. ドメイン知識
- 金融市場の取引原理
- 技術分析指標（RSI、MACD、移動平均等）
- 投資組合管理
- アルゴリズム取引戦略
- リスク評価と軽減策
- バックテスティングとパフォーマンス分析

## 使用シナリオ

### 機能実装の例

#### トレンドフォロー戦略の実装
```
トレンドフォロー戦略のエンジンを実装してください。20日EMAと50日EMAのクロスオーバーをシグナルとして使用し、
ストップロスはATRベースで設定し、テイクプロフィットはリスク報酬比2:1とします。
```

#### 取引所APIの統合
```
BinanceのWebSocket APIからBTC/USDTのリアルタイム価格データを取得するコンポーネントを実装してください。
再接続ロジックとデータキャッシングを含めてください。
```

### バグ修正の例

#### オーダーエンジンのエラー修正
```
orders.jsファイルのcreateOrderメソッドで、価格が0以下の時にエラーが発生しないバグを修正してください。
また、数量の精度を小数点以下8桁に制限するようにしてください。
```

### コードレビューの例

#### 取引戦略の改善
```
trendFollowingStrategy.tsのコードをレビューし、パフォーマンスと可読性を改善してください。
また、リスク管理ロジックの欠点を指摘して修正提案をしてください。
```

## 使用手順

### 1. モードの選択
1. VSCodeの左下隅にあるモード選択ボタンをクリック
2. "ULT Trading Platform Expert"を選択

### 2. 基本操作
- **コマンドパレット**：`Ctrl+Shift+P` → "Kilo"で機能を選択
- **エディタ内操作**：コードを選択 → 右クリック → "Kilo"機能
- **パネル操作**：左側バーのKiloアイコンをクリック

## キーボードショートカット

## ベストプラクティス

1. **明確な要求**：何を実現したいかを具体的に説明
2. **コンテキストの提供**：対象ファイルや関連する機能を明記
3. **分割実行**：大きな機能は小さなタスクに分割
4. **テストの重視**：機能実装後に必ずテストを実行
5. **リファクタリング**：定期的にコードの品質を改善
