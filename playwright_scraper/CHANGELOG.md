# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Playwright Web Scraper
- Browser automation with Playwright
- Login automation with configurable selectors
- Robust selectors with explicit waits
- Pagination handling
- Retry logic with exponential backoff
- Data export to JSON and CSV formats
- Rotating file logging
- CLI with argparse
- Request interception (block images, stylesheets, JavaScript)
- Viewport configuration
- Configuration file support (JSON/YAML)
- Environment variable support
- Custom exception hierarchy
- Comprehensive test suite with pytest
- Docker support (multi-stage builds)
- Docker Compose configuration
- GitHub Actions CI/CD pipeline
- Type hints throughout codebase
- mypy configuration
- Black, isort, flake8 code formatting
