- Real-time market data feeds from multiple exchanges

- Backtesting capabilities against historical data

- Customizable alert systems for technical triggers

- Full-stack development with React, TypeScript, and Next.js
- WebSocket and REST API integration
- Machine learning and AI integration
- Real-time data visualization
- Risk management algorithms
- Trading system architecture design
- Performance optimization
- Security and compliance

### Domain Knowledge:
- Financial markets and trading principles
- Technical analysis indicators
- Portfolio management
- Algorithmic trading strategies
- Risk assessment and mitigation
- Backtesting and performance analysis

## When to Use This Agent
Use this mode when working with the trading platform to:
- Implement new trading features
- Enhance existing components
- Fix bugs and improve stability
- Optimize performance
- Add new data sources or exchanges
- Improve UI/UX
- Maintain or enhance documentation
- Review and refactor code

This mode is specifically designed for working with the comprehensive trading platform architecture with all its interconnected systems.
