# Ult Trading Platform Implementation Roadmap

このドキュメントは、プロジェクトの進化における主要なマイルストーンと実装計画を定義します。

## 序文：現在のステータス
現在、プロジェクトは以下の基盤を確立しています。
- **ML予測エンジン**: RF, XGBoost, LSTMによるアンサンブル学習とATRボラティリティ調整。
- **UIアーキテクチャ**: Next.js (App Router) + TypeScript による高速でモジュール化されたUI。
- **検証基盤**: バックテストエンジンとデータアグリゲーター。

---

## 🚀 フェーズ 1：信頼性と堅牢性の確立（短期：1〜2週間）
新機能の安定化と、実運用に耐えうるエラー耐性の構築。

- [ ] **高度なエラーハンドリング**
  - API制限時のリトライ戦略と指数バックオフの実装。
  - データ欠損時の線形補間ロジックの導入。
- [ ] **包括的テストスイートの構築**
  - `ml-prediction` の全関数に対するカバレッジ 90% 以上の達成。
  - Playwright による主要ユーザーフローの E2E テスト自動化。
- [ ] **UI/UX の日本語化と最適化**
  - 残存する英語 UI（OrderPanel, Signal reasoning 等）の完全日本語化。
  - 日本市場の制限（呼値、制限値幅）を注文パネルに反映。

## 🤖 フェーズ 2：自律型トレーディングへの進化（中期：1ヶ月）
「ツール」から、意思決定をサポートする「パートナー」への進化。

- [ ] **仮想自動売買（Auto-Pilot）モード**
  - 高確度シグナル発生時に自動で仮想ポジションを構築するエンジン。
  - リアルタイムでの P&L トラッキングと履歴保存。
- [ ] **インテリジェント資金管理**
  - ケリー基準や固定リスク率（2%ルール等）に基づく自動ポジションサイジング。
- [ ] **マルチタイムフレーム・ハイブリッド分析**
  - 日足のトレンドと 5分足のモメンタムを組み合わせたシグナル精度の向上。
- [ ] **リアルタイム・シグナル通知**
  - ブラウザプッシュ通知、および Discord/Slack 連携。

## 📊 フェーズ 3：高度な分析とカスタマイズ（長期：2ヶ月〜）
プロフェッショナルな投資判断を支える高度な機能拡張。

- [ ] **ポートフォリオ相関分析エンジン**
  - 保有銘柄間のリスク相関（ベータ値、セクター集中度）の可視化。
  - AI による分散投資最適化のアドバイス。
- [ ] **マクロ経済データの統合**
  - ドル円為替、米国債利回り、主要指数（N225/SPX）を ML モデルの素性に追加。
- [ ] **カスタム戦略ビルダー**
  - ノーコードでテクニカル条件を組み合わせて戦略を作成・検証できる GUI。
- [ ] **エージェント・ラーニングの強化**
  - 過去の予測外れパターンを学習し、動的にモデル重みを調整するメタ学習層。

---

## 📈 優先順位と次のアクション
1. **信頼性向上**: データ欠損時のクラッシュ防止（最優先）。
2. **自動化の開始**: Auto-Pilot シミュレーターのプロトタイプ実装。
3. **UX 改善**: 英語表記の完全払拭。
